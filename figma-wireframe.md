## FIGMA SHORTCUTS REFERENCE

| Action | Shortcut |
|--------|----------|
| Frame | F |
| Rectangle | R |
| Text | T |
| Group | Cmd/Ctrl + G |
| Auto Layout | Shift + A |
| Component | Cmd/Ctrl + Alt + K |
| Variant | Cmd/Ctrl + Alt + V |
| Prototype Tab | Tab key (in Figma) |

## COLOR SWATCHES (For Figma Variables)

```
Primary Colors:
- Indigo-600: #4F46E5
- Purple-600: #9333EA  
- Pink-500: #EC4899

Gradients:
- Brand: #4F46E5 → #9333EA → #EC4899
- Primary: #4F46E5 → #9333EA

Backgrounds:
- Light: #F8FAFC
- Dark: #0F172A
- Card Light: #FFFFFF
- Card Dark: #1E293B

Text:
- Primary: #0F172A
- Secondary: #475569
- Muted: #94A3B8
```


| Card | Shadow md | Shadow lg, Y -2px | Shadow md | - |
| Nav Link | Color #475569 | Color #4F46E5 | Underline | - |

## NOTES

1. Use Auto Layout for all containers
2. Create Variants for Light/Dark mode
3. Use Component Properties for interactive states
4. Set up scrolling in Prototype mode
5. Add smart animate for hover transitions
